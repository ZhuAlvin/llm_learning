# Module 6: 高级训练技术 (Advanced Training) - 学习手册

> 📖 **本指南与 README 的关系**：README 提供模块概览和快速导航，本学习指南则提供详细的学习路径、深度解析和实践建议。建议先阅读 README 了解模块全貌，再按本指南深入学习。

## 1. 本模块解决什么问题

你将面对训练阶段最现实的三类问题：

- 模型训练不稳，loss 波动或不收敛
- 单机资源不够，训练速度跟不上业务节奏
- 显存与吞吐冲突，无法同时兼顾成本和效率

统一业务主线：`电商客服模型在活动高峰前的快速迭代训练`。

## 5. 学习路径

### 路径 A：瓶颈修复优先（6-8 小时）

`01（核心） -> 03（核心） -> 02（重点阅读）`

`01 -> 02 -> 03`

在路径 B 基础上增加：

- 构建训练资源预算模型

---

## 6. 学习曲线平缓策略

1. 先稳住优化基础，再做分布式扩展。
2. 先在小模型验证优化策略，再迁移到大模型。
3. 每次优化都同时看“效果、速度、显存”三指标。

---

## 7. 模块验收标准

- 能独立完成一次训练稳定性排障。
- 能解释至少两种并行策略的适用边界。
- 能提交一份显存-吞吐-效果三维对比结果。
- 能给出下一轮训练配置建议并说明依据。

---

## 8. 常见卡点与应对

- 卡点：吞吐提升但精度下降。  
  应对：检查混合精度与梯度缩放配置，增加精度回归测试。
- 卡点：多卡训练偶发中断。  
  应对：补齐 checkpoint 与自动重启策略。
- 卡点：优化策略太多无从下手。  
  应对：按优先级执行：先稳定性，再速度，后成本。
