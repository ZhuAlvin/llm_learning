# Module 8: 实际应用 (Applications) - 学习手册

> 📖 **本指南与 README 的关系**：README 提供模块概览和快速导航，本学习指南则提供详细的学习路径、深度解析和实践建议。建议先阅读 README 了解模块全貌，再按本指南深入学习。

## 1. 本模块解决什么问题

本模块聚焦“把模型能力变成用户价值”，核心问题是：

- 如何用 RAG (Retrieval-Augmented Generation, 检索增强生成) 降低幻觉并提供可追溯答案？
- 如何让 Agent (AI Agent, 智能体) 在可控范围内执行任务？
- 如何通过前端交互把系统体验做成可用产品？

统一业务主线：`电商客服智能助理产品化`。

## 6. 学习曲线平缓策略

1. 先解决“答对”问题，再解决“会执行”问题，最后优化“体验与运营”。
2. 每一章都保留最小可演示系统，避免概念堆叠。
3. 指标分层看：技术指标与业务指标同时追踪。

---

## 8. 常见卡点与应对

- 卡点：RAG 召回高但答案仍偏。  
  应对：检查上下文拼接与提示词约束，而非只调检索器。
- 卡点：Agent 调用工具不稳定。  
  应对：增加参数校验、超时、重试和白名单控制。
- 卡点：前端感觉“慢”。  
  应对：优先优化首字延迟并提供可见进度反馈。
