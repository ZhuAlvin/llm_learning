# Module 2: 模型演进 (Evolution) - 学习手册

> 📖 **本指南与 README 的关系**：README 提供模块概览和快速导航，本学习指南则提供详细的学习路径、深度解析和实践建议。建议先阅读 README 了解模块全貌，再按本指南深入学习。

## 1. 本模块解决什么问题

你会从序列建模的失败案例出发，回答：

- 为什么 RNN (Recurrent Neural Network, 循环神经网络) 在长文本里容易丢信息？
- 为什么 Attention (Attention Mechanism, 注意力机制) 能提高关键信息利用率？
- 为什么 Seq2Seq (Sequence-to-Sequence, 序列到序列) 是任务闭环的关键？

统一业务主线：`电商客服多轮会话理解与回复生成`。

## 6. 学习曲线平缓策略

1. 先接受“RNN 会失败”的事实，再学 LSTM (Long Short-Term Memory, 长短期记忆网络) 的门控思想。
2. 再用 Attention 解决信息瓶颈，不急于一次吃透所有变体。
3. 最后把模型组装成 Seq2Seq 闭环，形成完整任务视角。

## 8. 常见卡点与应对

- 卡点：维度错误频发。  
  应对：每个前向步骤都打印张量形状并写在注释中。
- 卡点：注意力图“看起来很乱”。  
  应对：先从短序列样本调试，再扩展到长序列。
- 卡点：生成结果重复或早停。  
  应对：对比 Teacher Forcing 比例与解码策略。
